# Конвертер валют с аналитикой

Веб-приложение для конвертации валют с аналитикой динамики курсов, Docker-контейнеризацией и базой данных PostgreSQL.

## Возможности

- Конвертация между различными валютами
- Отображение текущих курсов валют
- Аналитика динамики изменения курсов с графиками
- Сохранение истории конвертаций в базе данных
- Шифрование данных в базе данных
- Docker-контейнеризация для простого развертывания

## Технологии

- HTML5, CSS3, JavaScript (ES6+)
- Chart.js для визуализации данных
- Node.js + Express для серверной части
- PostgreSQL для хранения данных
- Docker и Docker Compose для контейнеризации
- Nginx для обслуживания статических файлов

## Установка и запуск

### Способ 1: С использованием Docker (рекомендуется)

1. Убедитесь, что у вас установлены Docker и Docker Compose
2. Клонируйте репозиторий
3. Создайте файл `.env` на основе `.env.example`
4. Запустите приложение:
